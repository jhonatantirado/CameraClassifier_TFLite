# TF Lite Android App

Este ejemplo es una aplicación móvil Android que usa la librería TensorFlow Lite y la cámara del teléfono para reconocer y clasificar más de 1000 objetos del mundo real

Fuente: https://www.tensorflow.org/mobile/tflite/demo_android

Repositorio original: https://github.com/tensorflow/tensorflow/tree/master/tensorflow/contrib/lite/java/demo/app

Pre requisitos:

Android 5.0 (API 21) o posterior para probar la aplicación móvil

Android Studio 3.1.3
